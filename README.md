# JavaScript Fetch API Basics

This project demonstrates the use of the **Fetch API** in JavaScript to retrieve data from a public REST API.

## Concepts Covered
- fetch() method
- Promises
- .then() chaining
- Asynchronous execution
- Handling JSON responses

## API Used
